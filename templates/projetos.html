Projetos

School API


 School API é um projeto desenvolvido em NodeJS, com utilização do framework NestJS e do banco de dados MongoDB. Esse projeto se trata de uma comunicação entre os responsáveis dos alunos e professores. Os responsáveis conseguem visualizar quais tarefas foram realizadas por seus dependentes e o desempenho deles. Essas informações são alimentadas pelos professores.

API Service


  API Service é um projeto de estudo para simular a comunicação de uma aplicação de um Service Desk. Esse projeto está sendo desenvolvido em Python, com a utilização do microframework Flask e integrando ao banco de dados MySQL. Nessa aplicação, o usuário consegue solicitar um serviço de TI e visualizar quais solicitações ele já realizou.
